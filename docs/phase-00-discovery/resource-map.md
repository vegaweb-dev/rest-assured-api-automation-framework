# Resource map
## Purpose
This document identifies the business resources exposed by the API and describes the relationships between them.
## Scope
This resource map covers the entities available in the stable GoRest environment used by this project.
## Overview
The stable GoRest environment exposes a single business resource: Users.
## Resources
| Resource | Description                                     | Identifier |
| -------- | ----------------------------------------------- | ---------- |
| Users    | Represents registered users managed by the API. | id         |
